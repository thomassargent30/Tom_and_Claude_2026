# Tom\_Claude\_work\_on\_book

Working drafts and notes produced in collaboration with Claude, exploring
themes in American fiscal history and optimal fiscal/monetary policy theory.

## Contents

| Document | Description |
|---|---|
| `Tom_Claude_Nobel_revised.tex` | Revised sections of a Nobel lecture incorporating state-vs-federal debt-assumption competition in the 1780s, drawing on Ferguson and Rothbard. |
| `state_defaults_1840s.tex` | State defaults and constitutional reform in the 1840s — Pennsylvania's default, New York's near-default, and the resulting restrictions on state borrowing. |
